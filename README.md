# Reporting SDK – Proof of Concept

This project is a Proof of Concept for integrating a reporting SDK into web frontends, developed as part of the JetBrains Internship Test Task.

## Tech Stack
- Kotlin
- Gradle
- JDK 17

## Project Structure
- `src/main/kotlin` – main implementation
- `src/test/kotlin` – tests

## How to Run
```bash
./gradlew test
